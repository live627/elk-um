Add custom buttons to the main menu within ElkArte

[font=Times New Roman][color=#cc6633][size=3][b]Introduction:[/b][/size][/color][/font]
[hr]
This allows custom buttons to be added to the main menu within ElkArte at will, complete with children and grandchildren menus (so, a main menu item, a dropdown and a follow-on dropdown)
[list]
[li]Setup this mod at Administration Center » Configuration » Ultimate Menu.[/li]
[li]Should "just work" with all other mods that add their own buttons, such as portals.[/li]
[li]Dynamic by design -- any menu button can change its text without a problem -- no special code required.[/li]
[/list]

[/list]
